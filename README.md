# handwritten-digit-recognition-cnn
A CNN-based handwritten digit recognition model trained on the MNIST dataset. The project demonstrates image preprocessing, convolutional neural network architecture, model training, and evaluation to accurately classify digits (0–9) using Python and TensorFlow/Keras.

## Dataset
The model is trained using the MNIST dataset which contains 70,000 images of handwritten digits (0–9).

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Seaborn

## Features
- Data preprocessing
- CNN model training
- Accuracy evaluation
- Visualization of results

## Model Architecture
The CNN model consists of:
- Convolution layers
- MaxPooling layers
- Dense layers
- Softmax output layer
